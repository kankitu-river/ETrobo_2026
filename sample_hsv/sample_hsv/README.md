# sample_hsv 使い方

カラーセンサの**HSVの値**を、USB接続でテラタームに2秒ごとに表示するプログラムです。
（走行しません／音も鳴りません。値を見るだけのツールです）

## 準備
- SPIKE と PC を **USBケーブル**でつなぐ


##  テラタームで値を見る
1. USBはつないだまま、**テラタームを起動**
2. 「新しい接続」→「シリアル」→ SPIKE の **COMポート**を選ぶ
   - どのCOMか分からなければ、Windowsの「デバイスマネージャー」で確認
3. 「設定」→「シリアルポート」→ スピードを **115200** にする 「やらなくてもいいかも」
4. SPIKE の**スタートボタン**を押す
5. 画面に流れればOK:
   ```
   === HSV log start ===
   HSV h=25 s=50 v=47
   HSV h=24 s=51 v=47
   ...
   ```
   `HSV` … 色相 h(0-359)・彩度 s(0-100)・明度 v(0-100)（2秒ごとに更新）
